飞行雪绒 LTS1.0.5pre2 更新日志
==============================

版本发布日期：2026-04-14
本次整理覆盖提交：2026-03-19 ～ 2026-04-15

`pre2` 这一轮不是单点功能更新，而是把 `3.18` 之后陆续落地的 AI、音乐、UI、对象系统、打包和文档改动统一收口到当前版本说明。

一、本次新增
------------
1. 元宝本地中转链路补齐。
   - 仓库内置 `services/yuanbao-free-api/` 源码与桌宠侧服务接入。
   - 控制面板支持启动本地元宝服务、弹出二维码登录面板、查看登录状态并退出登录。
   - OpenAI 兼容请求链路已补齐元宝参数与图片上传协同逻辑。

2. 新增启动与分发相关控制项。
   - 控制面板新增“启动时创建快捷方式”开关。
   - 新增绿色资源包打包脚本，便于携带模型与浏览器资源离线分发。

3. 控制面板新增“桌宠更新”标签页。
   - 提供“检查新版本”“同步开发版”两个入口。
   - 现阶段主要完成界面与交互占位，方便后续继续接线。

4. 新增 CMD 终端窗口模块。
   - 支持独立终端窗口、命令历史、加载动画、ANSI 清理与编码回退。
   - `/命令` 路由可直接转入 CMD 窗口执行，托盘也已补充入口。

5. 新增雪球系统。
   - 新增雪球对象、雪球管理器、拖拽投掷/冻结/寿命逻辑。
   - 新增雪球破碎与落地粒子、音效、控制面板配置项与托盘/命令接入。

6. 新增 GSV 语音控制与缓存管理。
   - 控制面板新增“自动启用 GSV 语音模块”开关。
   - 新增 GSV 缓存数量滑块（`1~128`，默认 `20`）和“打开缓存文件夹”按钮。
   - 生成的语音统一写入 `resc/user/temp/gsv_voice/`。

二、本次调整
------------
1. 音乐多源底层继续整理。
   - QQ / 网易云 / 酷狗搜索与播放入口继续统一到 `lib/script/music/`。
   - 音乐登录、鉴权、路由与服务抽象继续向统一结构收拢。

2. 元宝登录与退出链路继续稳定化。
   - 登录状态持久化逻辑进一步完善。
   - 退出流程与关闭链路已做收口，减少元宝服务和相关窗口残留。

3. GSV 语音模块开关语义收紧。
   - 开启时允许预热并允许 AI 文本转语音。
   - 关闭时既不预热，也不处理按需 AI 文本语音请求。
   - 缓存目录改为统一直存，不再为每条语音单独建文件夹。

4. 打包链路继续加固。
   - 普通包和绿色包都改为从 `config/version_info.py` 读取版本号。
   - 打包脚本会继续排除运行时目录，并对敏感配置做脱敏处理。
   - 部分大型资源改为随桌宠目录分发，降低因网络问题导致依赖未装全的概率。

5. 文档与资料舱已按当前实现同步重写。
   - `README.md`
   - `CHANGELOG.md`
   - `CONTRIBUTING.md`
   - `RELEASING.md`
   - `doc/` 下中文说明文档
   - `AA使用必读.html`

三、本次修复与收口
------------------
1. 修复并稳定了元宝面板、浏览器登录与登录态持久化相关问题。
   - 改善元宝登录视觉反馈过慢、登录异常与登录缓存偶发不落盘的问题。
   - 持续收敛元宝回复链路，减少异常胡言乱语。

2. 修复控制面板与相关 UI 的若干显示问题。
   - 修复 Ollama 模型列表偶发无法展开。
   - 修复控制面板打开时可能出现的显示异常。
   - 修复控制面板拉海洛字体水印显示不正确的问题。

3. 修复 GSV 和退出流程的收尾问题。
   - 修复 GSV 服务后台未正确关闭的问题。
   - 持续优化退出链路，缓解退出时黑屏问题。

4. 修复绿色包与发布包可能夹带敏感配置、运行时状态的问题。

5. 进一步将运行时产物与源码提交边界拆开：
   - `config/music/volume.json`
   - `config/user_scale.json`
   - `services/storage_state.json`
   - `resc/user/`
   - `logs/`

6. 控制面板标签栏边框、高亮样式和更新页布局已同步整理。

四、当前版本状态
----------------
- 主程序入口：`lib/core/qt_desktop_pet.py`
- 应用编排：`lib/script/main.py`
- AI：OpenAI 兼容 API / Ollama / YuanBao-Free-API / 规则回复
- 语音：GSV 文本转语音 + Vosk 本地识别
- 音乐：QQ / 网易云 / 酷狗多源入口
- UI：控制面板、桌宠更新页、CMD 终端、托盘入口已进入当前版本结构

五、说明
--------
- `pre2` 仍属于预发布版本，但当前条目已经补齐 `3.18` 之后的主要提交内容。
- 若后续继续沿用 `pre2` 版本号迭代，更新日志应继续在此基础上增补，而不要再回退为零散待办列表。
