当前工程状态
- 主入口：`lib/core/qt_desktop_pet.py`
- 应用编排：`lib/script/main.py`
- 核心机制：PyQt5 + EventCenter + 动态插件发现
- AI 模式：OpenAI 兼容 API / Ollama / YuanBao-Free-API / 规则回复
- 语音能力：GSV 文本转语音 + Vosk 本地识别
- 音乐能力：QQ / 网易云 / 酷狗多源搜索与统一播放入口
- 运行时数据：`resc/user/`、`logs/`、共享配置目录默认不进入 Git
