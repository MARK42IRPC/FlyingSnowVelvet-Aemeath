飞行雪绒粒子效果说明（LTS1.0.5pre3）

一、总体说明
- 事件入口：`EventType.PARTICLE_REQUEST`
- 渲染层：`lib/core/qt_particle_system.py`
- 粒子脚本发现：`lib/script/practical/manager.py`
- 自动扫描目录：`lib/script/practical/*_particle.py`

当前已注册粒子：16 个

三、统一事件格式
```python
Event(EventType.PARTICLE_REQUEST, {
    "particle_id": "scatter_fall",
    "area_type": "point",
    "area_data": (x, y),
})
```

区域数据格式
- `point`: `(x, y)`
- `rect`: `(x1, y1, x2, y2)`
- `circle`: `(x, y, radius)`
